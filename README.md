Учебная Виртуальная Машина — Вариант 7

Проект реализует стековую виртуальную машину, ассемблер и интерпретатор. Поддерживает выполнение программ, написанных на специально спроектированном языке ассемблера в формате YAML
Дамп памяти сохраняется в формате XML
Размер команды: 3 байта
Поля команды: `A` (Опкод, 6 бит), `B` (Операнд, 13 бит).

Проект включает кроссплатформенное GUI-приложение и консольные утилиты.


| :--- | :--- | :--- | :--- |
| **LOAD_CONST** | 32 / 0x20 | `value` | **Загрузка константы.**<br>Кладет значение `value` на вершину стека. |
| **READ_MEM** | 54 / 0x36 | `addr` | **Чтение из памяти.**<br>Читает значение из памяти по адресу `addr` и кладет его на стек. |
| **WRITE_MEM** | 52 / 0x34 | `offset` | **Запись в память.**<br>Снимает со стека **Базовый адрес**, затем снимает **Значение**.<br>Записывает *Значение* в память по адресу `Базовый адрес + offset`. |
| **SQRT** | 31 / 0x1F | `addr` | **Унарная операция.**<br>Читает значение из памяти по адресу `addr`, вычисляет его целочисленный квадратный корень и кладет результат на стек. |

### Пример программы  в YAML
program:
  - cmd: LOAD_CONST
    value: 64       # Положить 64 на стек
  - cmd: LOAD_CONST
    value: 0        # Положить адрес 0 на стек
  - cmd: WRITE_MEM
    offset: 0       # Записать 64 в память[0]
  - cmd: SQRT
    addr: 0         # Вычислить sqrt(Mem[0])
