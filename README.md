Учебная Виртуальная Машина — Вариант 7

Проект реализует стековую виртуальную машину, ассемблер и интерпретатор. Поддерживает выполнение программ, написанных на специально спроектированном языке ассемблера в формате YAML

Дамп памяти сохраняется в формате XML

Размер команды: 3 байта

Поля команды: `A` (Опкод, 6 бит), `B` (Операнд, 13 бит).

Проект включает кроссплатформенное GUI-приложение и консольные утилиты.

LOAD_CONST - Кладет значение на вершину стека.
READ_MEM - Читает значение из памяти по адресу кладет его на стек.
WRITE_MEM - Записывает значение в память по адресу Базовый адрес + сдвиг
SQRT - Читает значение из памяти по адресу, вычисляет его целочисленный квадратный корень и кладет результат на стек.

### Пример программы  в YAML
```
program:
  - cmd: LOAD_CONST
    value: 64       # Положить 64 на стек
  - cmd: LOAD_CONST
    value: 0        # Положить адрес 0 на стек
  - cmd: WRITE_MEM
    offset: 0       # Записать 64 в память[0]
  - cmd: SQRT
    addr: 0         # Вычислить sqrt(Mem[0])
```

### Установка УВМ на Linux/Windows
Откройте терминал и выполните команды:

```
git clone https://github.com/VintikG/uwm-crosspalatform-7.git

cd uwm-crosspalatform-7
```

### Активация окружения для Windows:

```
python -m venv venv

.\venv\Scripts\activate

### Активация окружения для Linux:

python3 -m venv venv 
source venv/bin/activate
```


### Запуск сборочного файла:

```
python build.py
```

### Запуск приложения на Windows:

```
dist\UVM_Variant7.exe

### Запуск приложения на Linux:

chmod +x dist/UVM_Variant7
./dist/UVM_Variant7
```
